# 팜스토리 프로젝트 (Farmstory Project)
> JSP 기반 커뮤니티형 농산물 쇼핑몰 웹 프로젝트

## 프로젝트 개요

- **카테고리**: 커뮤니티형 농산물 쇼핑몰
- **주요 기능**: 회원가입, 장바구니, 게시판, 관리자 페이지 등
- **개발 기간**: 2025.05 ~ 2025.06
- **참여 인원**: 3명 (프론트엔드 1, 백엔드 2)



## 기술 스택

- Java / JSP / Servlet / JDBC  
- HTML / CSS / JavaScript / Thymeleaf  
- MySQL / Apache Tomcat  
- Eclipse IDE / Maven

## 주요 기능

### 👤 사용자 기능
- 회원가입, 로그인, 로그아웃
- 상품 목록 / 카테고리 / 상세보기
- 장바구니 담기, 주문하기

### 📞 고객지원
- 공지사항
- Q&A, FAQ

### 🛠 관리자 기능
- 상품 등록, 수정, 삭제
- 회원 관리, 게시글 관리
